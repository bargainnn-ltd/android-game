## Couple Games

Android party game built with Jetpack Compose.

### Future development

- **Hub menu button**: Side sheet with links to Settings, How to Play, feedback, and legal pages.
- **Hub notifications button**: Activity feed for recent sessions, unlocks, and update tips.
- **Saved quick link & bottom-nav Saved**: Dedicated screen listing favorite prompts/questions with filters and management.
- **Reset session action in Settings**: One-tap way to clear current in-progress session state.
- **Decks bottom-nav item**: Deck browser to enable/disable decks by category and preview cards.
- **Profile bottom-nav item**: Lightweight profile with nickname, preferred intensity, and basic stats.
- **Leaderboard & Play bottom-nav items**: Leaderboard of player/session stats and a central “Play” hub for quick game starts.
